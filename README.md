
================================================================================
RESISTOR MATCHER
================

Инструмент для анализа и подбора комбинаций резисторов из заданной базы данных.

====================

3. Популярные команды
4. Параметры командной строки
5. Формат Excel-файла
6. Топологии соединений
7. Структура проекта
8. Требования
9. Автор
10. ВОЗМОЖНОСТИ
    ================================================================================

- Полный перебор комбинаций (размещения A(n,k)) для 1-4 резисторов
- 17 топологий: 10 для 4 резисторов, 4 для 3, 2 для 2, 1 для 1
- Расчёт точности сборки для каждой топологии
- Удаление дубликатов (отбрасываются комбинации с одинаковым набором номиналов)
- Оптимизированный поиск через TopologySearcher
- Фильтрация базы по соседним номиналам для ускорения
- Поиск по номиналу с выводом 10 ближайших слева/справа
- Объединённый поиск по всем топологиям (1-4 резистора)
- График локальной плотности точек ряда
- Оценка сложности (прогноз времени и памяти)

3. ПОПУЛЯРНЫЕ КОМАНДЫ
   ================================================================================

3.1. Базовая команда — объединённый поиск по всем топологиям
------------------------------------------------------------------------------------------------------------

# Поиск комбинаций из 1, 2, 3 и 4 резисторов (все этапы последовательно)

# Вывод: отдельные таблицы для каждого этапа + итоговая объединённая таблица

  python -m src.cli.commands.combined_search --nominal 89.341 --n-results 5

3.2. Раздельный поиск по количеству резисторов
-----------------------------------------------------------------------------------

# 1 резистор

  python -m src.cli.main --nominal 89.341 --optimized --auto-range --no-4 --no-3 --no-2 --1 --n-results 5

# 2 резистора

  python -m src.cli.main --nominal 89.341 --optimized --auto-range --no-4 --no-3 --no-1 --2 --n-results 5

# 3 резистора

  python -m src.cli.main --nominal 89.341 --optimized --auto-range --no-4 --no-2 --no-1 --3 --n-results 5

# 4 резистора

  python -m src.cli.main --nominal 89.341 --optimized --auto-range --no-3 --no-2 --no-1 --4 --n-results 5

3.3. Поиск по всем топологиям (с автоматическим выбором)
---------------------------------------------------------------------------------------------------

# Ищет по всем 17 топологиям, выводит 5 лучших результатов для каждой

  python -m src.cli.main --nominal 89.341 --optimized --auto-range --n-results 5

3.4. Оценка сложности (рекомендуется перед первым запуском)
----------------------------------------------------------------------------------------------------------

  python -m src.cli.main --estimate

3.5. Полный анализ с графиком плотности
---------------------------------------------------------------------

  python -m src.cli.main

3.6. Поиск по диапазону
---------------------------------------

  python -m src.cli.main --mode range --min 85 --max 95 --optimized --auto-range --n-results 10

4. ПАРАМЕТРЫ КОМАНДНОЙ СТРОКИ
   ================================================================================

4.1. Общие параметры
----------------------------------

  file                          Путь к Excel-файлу (по умолчанию: db/resistor.xlsx)
  --precision, -p               Точность округления результатов (по умолчанию: 6)
  --max-repeats, -r             Максимум экземпляров одного номинала (по умолчанию: 4)

4.2. Режимы поиска
------------------------------

  --mode, -m                    nominal (по номиналу) или range (по диапазону)
  --nominal, -n                 Целевое сопротивление (для --mode nominal)
  --min, -min                   Нижняя граница (для --mode range)
  --max, -max                   Верхняя граница (для --mode range)
  --n-results, -nr              Количество результатов на топологию (по умолчанию: 10)

4.3. Оптимизация
---------------------------

  --optimized, -opt             Использовать оптимизированный поиск (по умолчанию ВКЛЮЧЁН)
  --full-scan, -fs              Использовать полный перебор (ВЫКЛЮЧАЕТ оптимизированный)
  --auto-range, -ar             Автоматически отключать фильтрацию для быстрых топологий
  --range-nominals, -rn         Соседние номиналы: '15' или '10,5' (по умолчанию: 15)
  --fast-only, -fo              Только быстрые топологии (2+2, 2+1, 1+1, 1)
  --topology, -t                Конкретные топологии через запятую: '4-1,4-5,3-1'
  --structure, -st              Структуры через запятую: '2+2,2+1'

4.4. Топологии (включение/отключение)
-----------------------------------------------------------------

  --4, --3, --2, --1            Включить схемы из 4/3/2/1 резисторов (по умолчанию все включены)
  --no-4, --no-3, --no-2, --no-1  Отключить соответствующие схемы

4.5. Визуализация
-----------------------------

  --graph, -g                   Построить график плотности
  --save, -s                    Папка для сохранения графика (по умолчанию: img/)
  --no-show                     Не показывать график

4.6. Детализация
---------------------------

  --details, -d                 Показывать полную детализацию всех значений
  --limit, -l                   Лимит строк детализации (по умолчанию: 100)

6. ТОПОЛОГИИ СОЕДИНЕНИЙ
   ================================================================================

6.1. 4 резистора (10 топологий)
-------------------------------------------------

  R1+R2+R3+R4                       четыре последовательно
  1/(1/R1+1/R2+1/R3+1/R4)           четыре параллельно
  (R1+R2+R3)||R4                    три последовательно, параллельно четвёртому
  (R1||R2||R3)+R4                   три параллельно, последовательно с четвёртым
  (R1+R2)||(R3+R4)                  две последовательные пары параллельно
  (R1||R2)+(R3||R4)                 две параллельные пары последовательно
  R1+(R2||R3)+R4                    последовательно-параллельно-последовательно
  ((R1||R2)+R3)||R4                 параллельно-последовательно-параллельно
  R1+(R2||(R3+R4))                  последовательный резистор и параллель двух ветвей
  R1||(R2+R3+R4)                    параллельный резистор и последовательная цепочка

6.2. 3 резистора (4 топологии)
------------------------------------------------

  R1+R2+R3                          три последовательно
  1/(1/R1+1/R2+1/R3)                три параллельно
  (R1+R2)||R3                       два последовательно, параллельно третьему
  (R1||R2)+R3                       два параллельно, последовательно с третьим

6.3. 2 резистора (2 топологии)
------------------------------------------------

  R1+R2                             два последовательно
  R1||R2                            два параллельно

6.4. 1 резистор (1 топология)
----------------------------------------------

  R1                                одиночный резистор
